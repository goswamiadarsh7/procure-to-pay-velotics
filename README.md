# Project 2 — Procure-to-Pay Cycle Execution: Velotics Inc.

## SAP Supply Chain Portfolio | Project 2 of 5

---

## 📋 Overview

Executed an end-to-end Procure-to-Pay (P2P) cycle in a live SAP S/4HANA Cloud environment — sourcing 50 BKR-100 Frames from approved supplier Steel & More Inc. for production order replenishment at Plant 1 US (Company Code 1710). The cycle was completed with a verified 3-way match and documented as a professional portfolio deliverable.

---

## 🎯 Objectives

- Execute a complete P2P cycle in live SAP S/4HANA Cloud from requisition to invoice settlement
- Demonstrate 3-way match verification (PO – Goods Receipt – Invoice)
- Document the process with SAP transaction screenshots and document numbers
- Provide process improvement recommendations

## 💡 Process Improvement Recommendations

1. **Purchase Info Records** — Maintain PIRs for frequently purchased materials to auto-populate pricing and source
2. **MRP-Driven Requisitioning** — Automate PR creation through MRP runs rather than manual creation
3. **Supplier Performance Tracking** — Implement supplier evaluation scorecards for delivery reliability and quality
